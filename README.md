Project Overview
This project is a simple, responsive hotel landing page built as part of the Simplotel Frontend Assignment.
It demonstrates core frontend concepts including HTML structure, CSS layout, Flexbox, CSS Grid, and responsive design using media queries.
The page contains a navigation bar, image sections, a call-to-action area, and a separate menu page.
